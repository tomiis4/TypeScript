## Controls
- [ ] Basic
   - `:q` exit
   - `:w <file-name>` save

- [ ] NORMAL mode
   - `h` left
   - `l` right
   - `j` down
   - `k` up

   - `o` new line (down)
   - `O` new line (up)
 
   - [ ] `e` end of the word
   - [ ] `b` start of the word
 
   - [ ] `d` delete line
